```bash
# pwd -> /path/to/this/dir
$ pip install -r requirements.txt
```

# ipython notebooks
`notebooks`以下にipython notebookのファイルが入っています  
`ipython notebook`コマンドでターミナルから起動してください

```bash
# pwd -> /path/to/this/dir/notebooks
$ ipython notebook
```
